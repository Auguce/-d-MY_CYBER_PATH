Key Project: Key Exchange using Neural Networks

Honours Year Computer Science Project, University of Liverpool, UK
(Sept 2022 - Jun 2023)

This project delved into the innovative Kanter-Kinzel-Kanter (K-K-K) protocol, which employs neural networks for secure key exchange. Under the guidance of Professor Alexei Lisitsa, I explored the theoretical foundations of the protocol, analyzed its resilience to attacks, and developed a Python-based prototype to simulate its operation in real-world scenarios.

My work included:

  • Designing a neural network structure with defined weight update rules to implement the K-K-K protocol.
  • Building and testing synchronization mechanisms between communication parties (Alice and Bob).
  • Simulating and evaluating the protocol’s robustness against genetic algorithm-based attacks.
  • Analyzing how different parameters affected synchronization speed and the likelihood of successful attacks.

Despite some limitations in fully implementing advanced attack methods within the project timeline, the research highlighted the potential of neural networks in secure communication while identifying areas for further optimization and future exploration.

Key Project: Cybersecurity Course

Graduate-Level Cybersecurity Course, Boston University, USA
(Feb 2024 - Jun 2024)

This graduate-level course provided a comprehensive hands-on approach to cybersecurity, covering essential topics such as network forensics, exploitation techniques, SQL injection, reverse engineering, and more. It was a transformative experience that allowed me to bridge theoretical knowledge with practical application, focusing on real-world tools and scenarios.

Course Highlights:

  • Gained expertise in tools like nmap, Wireshark, tcpdump, and gdb.
  • Developed skills in network forensics, reverse engineering, and exploitation techniques.
  • Designed and implemented exploits, reverse shells, and payloads to simulate offensive security scenarios.

Selected Challenges

  • Network Forensics: Conducted packet analysis using Wireshark and tcpdump to identify compromised credentials, network anomalies, and hidden data.
  • Exploitation: Crafted buffer overflow and remote exploitation payloads to gain unauthorized access to systems and escalate privileges.
  • SQL Injection: Used SQL injection techniques to extract sensitive information and bypass authentication mechanisms.
  • Reverse Engineering: Decompiled Java applications and analyzed ELF binaries to uncover hidden logic and retrieve hardcoded secrets.
  • File Infector Virus: Designed a self-replicating Linux ELF virus, preserving original file functionality while executing malicious payloads.

Reflection: This course solidified my interest in penetration testing and offensive security. The hands-on challenges were instrumental in developing practical skills and provided invaluable insights into real-world cybersecurity scenarios.

Group Project: Log4Shell Vulnerability Scanner

Graduate-Level Cybersecurity Course, Boston University, USA
(Feb 2024 - Jun 2024)

Collaborated on a team project to develop a **custom network scanner** aimed at detecting the **Log4Shell vulnerability** (CVE-2021-44228). The scanner addressed limitations in existing tools like Fullhunt by introducing innovative features such as third-party service analysis and advanced payload customization.

Features:

  • Implemented scanning for LDAP, RMI, DNS, and CORBA-based JNDI lookup vulnerabilities.
  • Developed functionality to identify and analyze third-party integrations on target servers.
  • Enhanced scanning efficiency with multithreading and rate-limiting mechanisms.

My Contribution: Focused on URL scraping, scanner enhancements, and bug fixes to ensure reliable detection and comprehensive reporting.
